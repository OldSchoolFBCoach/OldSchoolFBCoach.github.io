Build 2108 Available - Create a Player & More

Announcement
August 07, 2021 (Link)

A new update (21-08) of Old School Football Coach is here! The new updates include the following:

New Features:

  • It is now possible to create and edit players. This can be accessed by editing a team through the league management menu and clicking a button to edit the team’s roster. From there you can select new options to add/edit/delete players.
  • It is now possible to set a team’s desired text font, similar to setting team colors. The football field will now display the home team’s mascot name in the end zones using the desired font.
  • Player names are now shown during the game action

Improvements

  • Players in the defense secondary are now smarter. If a play utilizes multiple safeties, they will try to do a better job covering different sides of the field and not bunch up. Secondary players will also now do a better job recognizing when they need to contest a catch. The latter improvement is actually more of a bug fix. When the ball was thrown by the offense, there was an error that made defenders ignore offensive players and thus always assume they could make an easy interception without worrying about an offensive player jumping in front of them to make the catch instead. This has been fixed so that defenders do a better job of recognizing when they should try to jump and “high point” a catch.
  • A consequence of making defensive players smarter, as described in the previous change, is that interceptions would occur a lot more often. To combat this, defender catching abilities have been reduced to bring the rate of interceptions back down.
  • Punters now try to kick balls at a slightly higher angle so that they don’t out-kick their coverage team as often.
  • Depth chart generation has been improved so that players aren’t placed out of their main position so much.
  • Added initial support for backwards compatible career and league saves. Up until this point, trying to use old save files with new builds of the game could lead to silent corruptions and crashes. Starting with save files generated by this build and going forward, save files will either be upgraded to be compatible with the new build, or you will be explicitly notified about the incompatibility.

Bug fixes

  • Fixed an issue that would prevent independent teams from generating a proper schedule in career mode. This bug would cause independent teams to only have 1 or 2 games on their schedule instead of the desired 12.
  • Fixed a few rare bugs that could lead to the game crashing

Happy coaching!
