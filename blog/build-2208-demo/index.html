Build 22-08 - Free Demo and More

Announcement
August 07, 2022 (Link)

A new update (22-08) of Old School Football Coach is here! The new changes include the following:

Free Demo!

Old School Football Coach now has a free demo available for download. The demo limits career mode to just 2 weeks of play, but is otherwise fully featured! You can now try the game out before you buy.

New Features

  • Each team now has 3 coaches, a head coach and two assistant coaches.
  • Head coaches and assistant coaches can now have abilities to help (or sometimes hurt) their team! These abilities, for example, might increase the rate of player development, help reduce the number of injuries, or help pull in new recruits.
  • Fumbles are now part of the game. Players have a new attribute, “Ball Security”, which determines how good they are at holding on to the ball. Low ball security can translate into an increased number of fumbles.
  • The game now keeps track of more player stats such as QB rating, pass completion percentage, dropped passes, average rushing yards, fumbles, FG percentage, and broken tackles.

Note: This build of the game is not backwards compatible with saves from previous versions.

Happy Coaching!
