Build 21-10 Available - Option Plays, Playbooks, & More

Announcement
October 03, 2021 (Link)

A new update (21-10) of Old School Football Coach is here! The new changes include the following:

New Features:

  • Teams now have head coaches associated with them that modify different aspects of the team.
    • Coaches now have their own playbook, meaning teams no longer have the same plays available to them. You can likewise set up your own playbook in career mode.
    • Coaches can set a desired offensive philosophy, such as “power run”, “quick pass”, and more. Offensive philosophy Impacts the AI coach play calling.
    • Coaches can set a desired balance between run and pass plays. This impacts the AI coach play calling.
    • Coaches can set their base formations. This impacts recruiting focus and AI coach playbooks.
  • There is now support for QB run option plays. Not only are option plays available to call during the game, but you can also now design your own QB run option style plays as well.
  • AI coaches will now adjust play calling based on player abilities. For example, they will pay attention to how strong of a leg their kicker has before attempting a long field goal.
  • More built-in plays have been added to the game.

Improvements / Tweaks / Bug Fixes

  • Defenders now have more of a random reaction delay when figuring out when the ball is run or thrown. This change slightly buffs offensive production.
  • When the ball is thrown to a receiver, other receivers will now attempt to block for them.
  • A player’s height is now properly used when determining collisions with the football. Previously the game assumed all players were 6ft tall.
  • In the play designer it’s possible to tell a receiver to stop once their route is completed, but they previously wouldn’t actually stop. This has now been implemented properly.
  • The play designer no longer allows you to give players bogus behaviors. These bogus behaviors were stand-in ideas for future additions to the game, but hadn’t been implemented. Thus players wouldn’t behave correctly if you gave them one of these behaviors.
  • Players no longer get fatigued when simulating plays in the play designer.
  • The stadium around the field has received some updated graphics.

Happy coaching!
